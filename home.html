What is Cloud Computing?

Cloud Computing is where a server that you connect to does the computing for you. Computing is a set of algorithms used to accomplish a task. Algorithms are sets of rules and steps to follow. Our definition of cloud computing is algorithms ran on a server to accomplish a task.

Some popular examples of Cloud Computing is Gmail, Drive, Drop Box, GeForce Now, and Wolfram Alpha.

Cloud Computing is more than just storage. It is programs that run on other servers. This can range from as simple as a calculator up to storage and live graphing.
