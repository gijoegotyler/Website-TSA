Advantages of Cloud Computing

Computer Requirements

The only requirement to run an application off the cloud is that you have an internet connection.

Easy For Collaboration

Your files aren't saved on a specific computers hard drive, but in a servers hard drives. This is good for the user because your files being stored on a server allows you to accesss the hard drive anytime, anywhere. Since, you can work anywhere off any computer, so can collaborators. Cloud allows for multiple people to be connected to a file at the same time, which makes collaboration significantly easier.

Security

Since, your files aren't saved on your computer you don't have to worry about someone stealing your computer and all your files. Also, if you drop your computer and break the hard drive disk, you don't have to worry because all your files are saved on a server.
