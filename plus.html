Advantages of Cloud Computing

Computer Requirements

The only requirement to run an application off the cloud is that you have an internet connection. Also, some cloud programs require faster or slower internet speeds.

Easy For Collaboration

Your files aren't saved on a specific computers hard, but in a servers hard drives. This is good for the user because it allows them to accesss the hard drive anytime, anywhere. Since, you can work anywhere off any computer, so can collaborators. Cloud allows for multiple people to be connected to file at the same time, which makes collaboration significantly easier.

Security

Since, your files aren't save on your computer you don't have to worry about someone stealing your computer and all your files. Also, if you drop your computer and break the hard drive disk, you don't have to worry because it is all saved on the cloud.
